◆ Funcionalidades

Tudo que você precisa para codar com IA

Do chat inteligente ao terminal integrado, cada detalhe foi pensado para maximizar sua produtividade com agentes de código.

Chat com Agentes de IA

Interface de conversação multi-thread com streaming em tempo real, renderização Markdown e suporte a anexar imagens (até 10 MB, 8 por turno).

Core

Diff de Código Integrado

Painel de diffs com syntax highlighting, árvore de arquivos alterados, toggle de word-wrap e visualização clara das mudanças propostas pela IA.

Core

Terminal Embutido

Terminal completo baseado em xterm.js com suporte a múltiplas instâncias, split, histórico de sessão, detecção de atividade e dimensões configuráveis.

Core

Dual Provider

Suporte nativo para Claude Agent (Anthropic) e Codex (OpenAI) com seleção de modelo, detecção de capabilities e modelos customizados.

Pro

Integração Git Completa

Branches, worktrees, checkout, status, commit, push, criação de PRs e ações empilhadas (commit + push + PR) com tracking de progresso e output de hooks.

Core

Multi-Projetos

Gerencie múltiplos projetos simultaneamente com bootstrapping, scripts com ícones dedicados (play, test, lint, build, debug), busca de arquivos e tracking de alterações.

Políticas de Aprovação

Controle granular de segurança: untrusted, on-failure, on-request ou never. Sandbox com modos read-only, workspace-write e full-access.

App Desktop

Desktop

Configurações Avançadas

Keybindings customizáveis, temas (system/light/dark), formato de horário, reasoning effort, fast mode, thinking toggle e contexto de janela.

Core

Worktrees Isolados

Crie workspaces Git isolados para cada thread de conversação, permitindo trabalhar em múltiplas features sem conflitos. Limpeza automática inclusa.

Novo

Comunicação em Tempo Real

Atalho Ação Contexto

Mod + Shift + N Novo chat local Global
Mod + O Abrir no editor favorito Global

◆ Arquitetura

Design modular e escalável

Monorepo com contratos compartilhados, orquestração event-driven e injeção de dependência via Effect Layers.

Pacotes do Monorepo

  • @t3/web — Aplicação React (SPA)
  • @t3/server — Servidor Node.js + WebSocket
  • @t3/desktop — Shell Electron
  • @t3/contracts — Tipos e schemas compartilhados

Padrões Arquiteturais

  • Effect-TS para efeitos gerenciados
  • Orquestração event-driven (projector/decider)
  • WebSocket bidirecional em tempo real
  • Injeção de dependência via Layers
  • Schema validation nos contratos
  • Sem barrel files (subpath exports)

Configuração e Estado

  • ~/.t3/keybindings.json
  • ~/.t3/settings.json
  • ~/.t3/state.sqlite
  • Variáveis de ambiente (T3CODE_*)
  • Bootstrap envelope seguro
